# Zomato Restaurant Data Analysis

## Project Overview

This project analyzes the Zomato restaurant dataset to identify trends in restaurant ratings, cuisines, locations, customer engagement, and pricing.

## Dataset

* Source: Zomato Dataset
* Records: 56,252 restaurants
* Features: Ratings, votes, cuisines, locations, cost, and restaurant details

## Data Cleaning

* Cleaned restaurant ratings (`clean_rate`)
* Cleaned cost column (`clean_cost`)
* Handled missing values
* Removed unrealistic cost outliers

## Visualizations

1. Rating Distribution Histogram
2. Top 10 Cuisine Types
3. Top 10 Restaurant Locations
4. Price vs Rating Scatter Plot
5. Correlation Heatmap
6. Cuisine Word Cloud

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* WordCloud
* Google Colab

## Key Findings

* Most restaurants have ratings between 3.5 and 4.2.
* North Indian cuisine is the most popular cuisine category.
* BTM, HSR, and Koramangala are major restaurant hubs.
* Ratings are moderately related to customer votes.
* Restaurant cost has only a weak relationship with ratings.
